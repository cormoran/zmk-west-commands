## できること

west マニフェストに 1 行追加するだけで、4 つのコマンドが `west` に現れます。2 つは
`zmk-config` ファームウェアのビルドを手軽にし、残り 2 つは **ハードウェアを 1 台も使わずに**
あなたのモジュールのテストを実行します。

<CardGrid stagger>
  <Card title="west zmk-build" icon="seti:cpp">
    `build.yaml` を読み取り、ターゲットを対話的に選び、フラッシュまで行う
    `west build` ラッパー。`--reset`・`--debug-jlink`・`--vscode` などの
    便利なショートカット付き。
  </Card>
  <Card title="west zmk-test" icon="approve-check">
    ZMK の native-posix ユニットテストを、環境変数を自動設定して実行。
    ディレクトリを指定し `-m .` で自分のモジュールを含めるだけ。
  </Card>
  <Card title="west zmk-renode-test" icon="rocket">
    実ファームウェア ELF を Renode エミュレータで起動し、boot + Studio の
    スモークテストを実行。BLE ペアリングまたは UART、J-Link も実機も不要。
  </Card>
  <Card title="west zmk-ble-test" icon="random">
    Split と Studio-over-BLE の結合テストを BabbleSim 上で実行し、出力を
    リポジトリ内スナップショットと照合。完全にソフトウェアのみ。
  </Card>
</CardGrid>
